Installation

Installing Python is easy. Download the Python to your computer and follow the screen instructions. Be sure to download the correct Python version for your operating system e.g. Linux, macOS, Windows, or other.

Download from Python.org/downloads

Installing Pandas can be different depending on your project. Some of the options are: install Pandas as part of Anaconda distribution or with Miniconda, which is a lighter version of Anaconda, PyPi, and other distributions. To help you decide the method of installation for your project, please visit the Pandas website link below.
